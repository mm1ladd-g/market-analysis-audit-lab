# پشتیبانی

[فارسی](SUPPORT.fa.md) · [English](SUPPORT.md)

پشتیبانی جامعه در حد توان است. پروژه مشاورهٔ مالی، حقوقی، انطباق، مجوز داده، سرمایه‌گذاری یا اضطراری ارائه نمی‌کند و زمان پاسخ تضمین‌شده ندارد.

## محل درخواست

- **استفاده یا تنظیمات:** GitHub Discussions در صورت فعال بودن؛ در غیر این صورت Issue پرسشی.
- **Bug قابل‌بازتولید:** Issue خطا با ورودی ساختگی یا حذف‌شده.
- **قابلیت جدید:** Issue پیشنهاد با توضیح اثر بر شواهد و ایمنی.
- **مستندات:** Issue مستندات با نام صفحهٔ هر دو زبان.
- **آسیب‌پذیری:** مسیر خصوصی [SECURITY.fa.md](SECURITY.fa.md)، نه Issue عمومی.
- **اصلاح ممیزی منتشرشده:** تماس با اپراتور همان ممیزی از مسیر درج‌شده در گزارش. نگه‌دارندگان متن‌باز، استقرار مستقل را کنترل نمی‌کنند.

## اطلاعات مفید برای بررسی

نسخه یا Commit، سیستم‌عامل، نسخهٔ Docker و Compose، دستور، تنظیمات پاک‌سازی‌شده، وضعیت مرحله و خطای حداقلی را ارائه دهید. کلید، Cookie، نام کاربر محلی، دادهٔ خصوصی مالک کانال، رونویسی کامل و ردیف بازار اختصاصی را حذف کنید. برای بازتولید Bug از نمونهٔ ساختگی استفاده کنید.

```bash
python -m audit_lab.cli doctor
python -m audit_lab.cli status
python -m unittest discover -s tests -v
```

## درخواست‌های خارج از پشتیبانی

نگه‌دارندگان نمی‌توانند حق قانونی شما نسبت به محتوا را تعیین کنند، معامله انتخاب کنند، نتیجهٔ ممیزی را تضمین کنند، سودآوری شخص را برآورد کنند، کلید افشاشده را بازیابی کنند یا خوراک بازار دارای مجوز ارائه دهند. درخواست آزار، افترا، افشای هویت، دست‌کاری تعامل یا دورزدن کنترل پلتفرم بسته خواهد شد.
