# مدل‌های OpenAI، جریان داده و هزینه

[فارسی](openai-models-and-cost.md) · [English](../en/openai-models-and-cost.md) · [مستندات](index.md)

OpenAI برای نمونهٔ ساختگی اختیاری است و فقط برای استخراج بدون حافظهٔ نهان یا امتیازدهی تفسیری در حالت API لازم می‌شود.

## جریان داده

استخراج ادعا، فرادادهٔ انتخاب‌شدهٔ ویدئو، هش رونویسی و تمام رونویسی مرجعِ شماره‌گذاری‌شدهٔ همان ویدئو را ارسال می‌کند. امتیازدهی، فرادادهٔ ویدئو، ادعاهای ساختاریافته و پنجره‌های ساختاریافتهٔ پیامد را می‌فرستد. پرامپت‌ها از مدل خروجی ساختاریافته با پشتوانهٔ Pydantic می‌خواهند. درخواست‌ها `store=false` دارند، اما اپراتور باید مستندات جاری پلتفرم OpenAI و شرایط حساب خود را بررسی کند و از این گزینه وعدهٔ نگهداری استنباط نکند.

محتوایی را که اجازهٔ پردازش آن را ندارید ارسال نکنید. رونویسی را ورودی غیرقابل‌اعتماد و مستعد تزریق پرامپت بدانید. اعتبارسنجی برنامه، شناسهٔ ویدئو و ادعا، مرز خطوط، پوشش گزیده، سازگاری امتیاز و نتیجه، وضعیت فعال‌شدن و کفایت داده را کنترل می‌کند.

## مدل‌ها

شناسهٔ دقیق مدل‌های در‌دسترس حساب خود را با `OPENAI_MODEL_CLAIM_EXTRACTION` و `OPENAI_MODEL_SCORING` تنظیم کنید. شناسهٔ مدل درخواستی و بازگشتی را ثبت کنید. در سابقهٔ بازتولید از واژهٔ «جدیدترین» استفاده نکنید. ارائه‌دهنده ممکن است یک مدل ثابت را بازنشسته کند؛ فایل پاسخ پذیرفته‌شده و هش آن را حفظ کنید.

## کنترل هزینه

مراحل پولی به `AUDIT_MODE=api`، کلید زمان اجرای `OPENAI_API_KEY` و `API_COST_ACKNOWLEDGED=true` نیاز دارند. با `--limit 1` آغاز کنید، مصرف را بررسی کنید و سپس هم‌زمانی را محافظه‌کارانه تنظیم کنید. متغیرهای اختیاری قیمت، هزینهٔ پاسخ پذیرفته‌شده را از توکن‌های ورودی، ورودی ذخیره‌شده و خروجی برآورد می‌کنند؛ فراخوانی ناموفق نیز ممکن است هزینه داشته باشد که مصرف ثبت‌شدهٔ محلی آن را نشان ندهد. پیش از هر ممیزی قیمت‌ها را در وب‌سایت رسمی OpenAI بررسی کنید.

## حافظهٔ نهان و اجرای دوباره

اثرانگشت حافظهٔ نهان به شواهد، پرامپت، طرح‌واره یا سیاست، مدل و تنظیمات استدلال وابسته است. `--force` عمداً حافظهٔ نهان معتبر را کنار می‌گذارد و ممکن است دوباره هزینه ایجاد کند. ورودی یکسان، متن یا قضاوت یکسان مدل را تضمین نمی‌کند؛ بازتولیدپذیری یعنی حفظ و راستی‌آزمایی فایل پذیرفته‌شده، نه وعدهٔ تولید دوبارهٔ قطعی.
