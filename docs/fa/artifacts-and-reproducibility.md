شناسهٔ مجموعه از تصویر ثابت منبع مشتق می‌شود، نه از یک عنوان تبلیغاتی. مرحلهٔ نهایی‌سازی، شناسه‌های ناسازگار مجموعه، مراحل ناقص، نسخه‌های قدیمی طرح‌واره یا سیاست، اختلاف دفتر تجمیعی با فایل هر ویدئو، هش شکستهٔ رونویسی یا منبع بازار، هش ناسازگار پیامد، اثرانگشت نامعتبر امتیازدهی و محاسبهٔ دوبارهٔ ناسازگار مخرج یا نتایج را رد می‌کند.

## دو معنای بازتولیدپذیری

**راستی‌آزمایی فایل‌ها** اندازه و مقدار SHA-256 را دوباره محاسبه می‌کند، سلامت ZIP را می‌آزماید و نشان می‌دهد بایت‌های بررسی‌شده با بستهٔ ثبت‌شده یکسان‌اند.

**بازتولیدپذیری اجرای محاسباتی** می‌پرسد آیا فراخوانی تازهٔ منبع، ارائه‌دهنده یا API نتیجه‌ای معادل تولید می‌کند یا نه. فرادادهٔ متغیر پلتفرم، تاریخچهٔ اصلاح‌شدهٔ بازار، مدل‌های بازنشسته، پاسخ‌های احتمالی، تغییر وابستگی‌ها و گذر زمان این امکان را محدود می‌کنند. راستی‌آزمایی هش، حقیقت، کامل‌بودن، قانونی‌بودن یا یکسان‌بودن اجرای دوباره را ثابت نمی‌کند.

## سابقهٔ بازتولید

شناسهٔ commit یا tag در Git، چکیدهٔ تصویر کانتینر، تنظیمات پاک‌سازی‌شده، زمان‌های دریافت به UTC، مرزهای منبع، دفتر حذف‌ها، ارائه‌دهنده/بازار/نماد/فاصلهٔ زمانی، هش‌های خام و عادی‌سازی‌شده، شناسهٔ مدل درخواستی و بازگشتی، تنظیمات استدلال، شناسهٔ پاسخ، مصرف توکن، هش پرامپت/طرح‌واره/سیاست، کلیدهای حافظهٔ نهان، آزمون‌ها، بازبین و هش ZIP نهایی را حفظ کنید.

هیچ کلید محرمانه، کوکی، نام کاربر محلی یا محتوای خصوصیِ غیرضروری منبع را وارد نکنید. راستی‌آزما باید بتواند `verify` و `verify-final` را بدون کلید API یا اتصال شبکه اجرا کند.

بستهٔ نهایی شامل `components/runtime_settings.public.json` با تنظیمات پاک‌سازی‌شده و نسخهٔ برنامه، هر دو دفتر بازبینی با زنجیرهٔ هش برای ممیزی عمومی، مانیفست انتشار فعال‌شده، تنظیمات بازبینی‌شدهٔ سیاست و هش آن‌ها، مانیفست عادی‌سازی‌شدهٔ منابع ارائه‌شده در بازه ـ در صورت کاربرد ـ و نسخهٔ قابل‌انتقال `components/market_evidence/` همراه مانیفست مسیر نسبی و هش است. این بسته `.env`، کلید API، متن راهنمای رونویسی یا مسیر مطلق اپراتور را کپی نمی‌کند.
