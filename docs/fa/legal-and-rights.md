# حقوق و مجوز منبع

[فارسی](legal-and-rights.md) · [English](../en/legal-and-rights.md) · [مستندات](index.md)

این راهنمای کاهش خطر است، نه مشاورهٔ حقوقی.

فقط محتوایی را وارد کنید که مالک آن هستید، برای پردازش آن مجوز دارید یا قانوناً حق استفاده دارید. عمومی بودن URL مجوز نیست. پروژه اجازهٔ دانلود، رونویسی، ذخیره، ارسال به هوش مصنوعی، تغییر، بازنشر یا انتشار شخص ثالث نمی‌دهد.

شرایط YouTube دسترسی خودکار و دانلود را جز در صورت اجازهٔ سرویس، رضایت قبلی یا قانون محدود می‌کند و API نیز الزام حریم خصوصی، انتساب، نگهداری و منع Scrape دارد. [شرایط YouTube](https://www.youtube.com/static?template=terms) و [سیاست توسعه‌دهنده](https://developers.google.com/youtube/terms/developer-policies) را بررسی کنید. تأیید Gate، مصونیت نیست.

Export مالک محتوا، Caption صاحب حق یا API مجاز را ترجیح دهید. لوگو، تصویر، Thumbnail، Font، دادهٔ بازار و Trademark مجوز جدا دارند. مجوز، منشأ، تاریخ، کاربرد و انقضا را ثبت کنید. مجوز Apache پروژه فقط کد و مستندات اصلی را پوشش می‌دهد.

ابزار توصیهٔ سرمایه‌گذاری، اثبات سود، گواهی تحلیل‌گر یا پیش‌بینی آینده نیست. از نتیجهٔ گمراه‌کننده یا افترا دوری، عدم‌قطعیت و حمایت مالی را اعلام و حق پاسخ فراهم کنید.
