# رونویسی و زبان

[فارسی](transcripts-and-language.md) · [English](../en/transcripts-and-language.md) · [مستندات](index.md)

رونویسی فقط هنگامی استفاده شود که گردآوری، پردازش، ارسال به مدل، نگهداری و انتشار موردنظر مجاز باشد. منبع ترجیحی، Caption متعلق به اپراتور، فایل صاحب حق یا Export مجاز است. Gate حقوقی، مجوز ایجاد نمی‌کند.

ترتیب `SUBTITLE_LANGUAGES` تعیین‌کننده است. دستی یا خودکار بودن، Tag زبان، قالب، زمان دریافت، فراداده و هش خام را حفظ کنید. نبود زیرنویس لازم یک حذف مبتنی بر شواهد است.

فایل خام را کنار متن UTF-8 نگه دارید. Manifest یک Sidecar هش‌شدهٔ `*.timing.json` نیز می‌سازد که هر خط مرجع را به زمان آغاز و پایان SRT/VTT وصل می‌کند؛ ورودی متن ساده زمان `null` صریح دارد. تکرار Cue و Markup را محافظه‌کارانه حذف کنید، اما دستور زبان، شرط، نفی، عدم‌قطعیت، عدد و معنا را «صیقل» ندهید. نسخهٔ خواندنی ویرایش‌شده فقط برای ارائه است.

گفتاربه‌متن مجوز دانلود صدا نیست. مرحلهٔ اختیاری `transcribe` پیش‌فرض خاموش است، فقط ویدئوهای داخل دامنه از کانال YouTube اعتبارسنجی‌شده را می‌گیرد و URL هر ویدئو را Canonical می‌کند. پردازش‌های رسانه‌ای کلید OpenAI را به ارث نمی‌برند و محدودیت حجم، مدت و زمان اجرا دارند. مدل/تنظیم، Segment زمانی و هش ورودی/خروجی ثبت و پیش‌فرض صوت خام حذف می‌شود. اصلاح انسان از خروجی مرجع جدا می‌ماند. رونویسی حاوی تزریق پرامپت ممکن است؛ شاهد است، نه دستور.
