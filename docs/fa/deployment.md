# استقرار

[فارسی](deployment.md) · [English](../en/deployment.md) · [مستندات](index.md)

حالت پشتیبانی‌شدهٔ پیش‌فرض localhost است. پیش از هرگونه دسترسی شبکه، نمونهٔ ساختگی و راستی‌آزمایی را محلی اجرا کنید.

</div>

```bash
docker compose up --build
```

<div dir="rtl" lang="fa">

Compose مسئولیت‌ها را جدا می‌کند: worker دارای profile با نام `audit-lab` فقط هنگام اجرای دستور صریح، تمام فضای کاری و کلیدهای محرمانهٔ تنظیم‌شدهٔ زمان اجرا را دریافت می‌کند؛ فرایند پیش‌فرض `audit-web` فقط `workspace/reports` را به‌صورت فقط‌خواندنی می‌بیند، فایل‌سیستم ریشهٔ فقط‌خواندنی دارد، فقط به localhost متصل می‌شود و کلید OpenAI دریافت نمی‌کند. Compose فقط مقدارهای عمومی و اعتبارسنجی‌شدهٔ `AUDIT_RELATIONSHIP_DISCLOSURE`، `CORRECTION_CONTACT` و مقدار اختیاری `CORRECTION_POLICY_URL` را که برای بررسی snapshot پذیرفته‌شده لازم‌اند به نمایشگر می‌فرستد و فایل `.env` را در کانتینر وب mount نمی‌کند. این دو سرویس را برای راحتی یکی نکنید. نمایشگر از شبکهٔ عادی Docker استفاده می‌کند تا درگاه منتشرشده در Docker Desktop و Linux یکسان کار کند؛ هرجا محدودیت خروجی شبکه لازم است، دیوارهٔ آتش میزبان یا سیاست شبکهٔ ویژهٔ استقرار را اعمال کنید.

اتصال میزبان را روی `127.0.0.1` نگه دارید. برای سرویس مشترک یا عمومی، یک reverse proxy یا tunnel بازبینی‌شده جلوی برنامه قرار دهید و TLS، احراز هویت، مجوز دسترسی، محدودیت نرخ و اندازه، headerهای امن، پاک‌سازی log، پایش، نسخهٔ پشتیبان و `CORRECTION_CONTACT` را اضافه کنید. از درگاهی غیرمعمول استفاده کنید، بی‌آن‌که دامنه یا حساب خصوصی را در مخزن کد ثبت کنید.

مخزن، پوشهٔ خانه، socket مربوط به Docker، فایل `.env` یا فضای کاری نامرتبط را به کانتینر وب متصل نکنید. شواهد خام و حافظهٔ نهان هوش مصنوعی را بیرون از ریشهٔ فایل‌های ثابت نگه دارید. حالت اشکال‌زدایی و فهرست پوشه‌ها را غیرفعال کنید. CORS، میزبان‌های مورداعتماد، خروجی شبکه و endpointهای گزارش را محدود کنید. بررسی سلامت باید فقط وضعیت را نشان دهد، نه تنظیمات یا شواهد را.

tunnel ابری به‌تنهایی مجوز دسترسی برنامه ایجاد نمی‌کند. حافظهٔ نهان، log دسترسی، نمایه‌سازی ربات، پیش‌نمایش پیوند و اثرهای جغرافیایی یا حریم خصوصی را بازبینی کنید. داشبورد عمومی یک رویداد انتشار است و باید فهرست کنترل حقوق، حریم خصوصی و انصاف را بگذراند.
